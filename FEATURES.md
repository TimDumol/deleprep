# Features Documentation

## 1. Writing Practice (Existing Feature)
The platform offers a core AI-driven writing practice module designed to help users prepare for the DELE A2 exam.

- **Task Selection:** Users can select between different types of tasks (e.g., Email/Message, Narrative/Opinion). Each task clearly identifies the grammar and vocabulary focus.
- **Dynamic Prompts:** The system generates personalized prompts with specific scenarios and bullet points, explicitly outlining the target skills (e.g., Pretérito Indefinido, Pretérito Imperfecto).
- **Writing Interface:** A clean, distraction-free text area that provides real-time word count tracking to ensure responses meet the DELE A2 requirements (e.g., 60-80 words).
- **AI Grading & Feedback:** Upon submission, the platform simulates an AI grading process that evaluates the text. It returns:
  - An overall verdict (Pass/Fail) and DELE Score (0-3).
  - Detailed, inline corrections showing the original error, the corrected version, and an explanation.
  - Skill tracking updates, explicitly showing which skills improved and which need more practice based on the submission.

## 2. Targeted Multiple-Choice Exams (New Feature)
To complement the free-form writing practice, a new multiple-choice exam feature has been introduced to drill down on specific weak points using spaced repetition concepts.

- **Dashboard Integration:** Accessible directly from the main dashboard alongside the writing tasks.
- **Personalized Generation:** The system analyzes the user's past performance and generates a custom, targeted multiple-choice exam focusing on their weakest skills and unexplored areas.
- **Interactive Exam UI:** Users proceed through a seamless interface to answer the generated questions. Each question clearly indicates the specific skill being tested.
- **Instant Grading & Spaced Repetition:** Once submitted, the exam is instantly graded. The feedback view provides:
  - An overall score out of the total questions.
  - Detailed review for each question, showing the user's answer, the correct answer, and an explanation.
  - **Spaced Repetition Tracking:** Explicit feedback on how the performance on each question affects the spaced repetition schedule (e.g., "Review in 7 days" for correct answers, "Review Tomorrow" for incorrect answers), reinforcing long-term retention.
